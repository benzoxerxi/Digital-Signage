# 📱 Android APK Configuration for Multi-Tenant SaaS

## How the Connection Works

Each user gets a **unique 9-digit connection code** (e.g. `847291536`) that identifies their account to the server. The APK uses this code instead of exposing internal user IDs.

**Benefits:**
- Easy for users to enter (9 digits, no login required on the display)
- Unique per account, randomly generated
- More secure than exposing database IDs
- Shorter than API keys

---

## 🔧 Setup Steps

### Step 1: Get Your 9-Digit Connection Code

1. Log in to the dashboard
2. Go to **Account** (in the menu)
3. Find your **Connection Code (for APK)** — a 9-digit number like `847291536`

### Step 2: Configure the APK

In your Android app settings, enter:

**Option A – Code only (recommended):**  
Configure the APK to use:
- **Server URL:** `http://YOUR_SERVER_IP:5000/api/playback/state`
- **Connection code:** Your 9-digit code (e.g. `847291536`)

**Option B – Full URL:**  
If the APK expects a full URL:
```
http://YOUR_SERVER_IP:5000/api/playback/state?code=847291536
```

Replace:
- `YOUR_SERVER_IP` with your server's IP or domain (e.g. `192.168.1.100` or `signage.example.com`)
- `847291536` with your actual 9-digit connection code

### Step 3: Video URLs

The server returns video URLs that include the code automatically:
```
/api/video/filename.mp4?code=847291536
```

The APK should use these URLs as returned — no extra configuration needed.

---

## 📡 API Endpoints (APK Use)

| Endpoint | Parameter | Description |
|----------|-----------|-------------|
| `/api/playback/state` | `code=XXXXXXXXX` | Poll for current playback state (9-digit code) |
| `/api/video/<filename>` | `code=XXXXXXXXX` | Stream video file |
| `/api/playback/next` | `code=XXXXXXXXX` | Get next video in playlist |

**Legacy support:** `user_id=N` is still accepted but the 9-digit code is preferred.

---

## ✅ Test Connection

Test in a browser or with curl:
```
http://localhost:5000/api/playback/state?device_id=test123&code=847291536
```

You should get a JSON response with playback info.

---

## 🔄 Regenerating Your Code

Contact support if you need a new connection code. Codes are permanent unless regenerated by an admin.

---

## 🛡️ Security Notes

- Keep your connection code private (like a password)
- Each code is unique to your account
- Only active subscribers can connect displays
- Devices are isolated per tenant
